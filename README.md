# Agir_MediaCR 📊

Sistema desenvolvido para gestão e cálculo de médias acadêmicas, focado em organização de dados e boas práticas de desenvolvimento Java. O projeto nasceu de uma necessidade prática e foi evoluído para seguir padrões profissionais de mercado.

## 🚀 Melhorias Recentes (Março/2026)
Recentemente, o projeto passou por uma refatoração completa para elevar o nível técnico:
- **Migração para Maven:** Implementação do gerenciamento de dependências e automação de build.
- **Java 21 LTS:** Atualização para a versão mais recente de suporte de longo prazo do Java.
- **Padrão de Pastas:** Reestruturação para o *Standard Directory Layout* do Maven (`src/main/java`).
- **Testes Unitários:** Adição de cobertura de testes com **JUnit 5** para garantir a integridade dos cálculos.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Java 21
- **Gerenciador de Build:** Maven
- **Framework de Testes:** JUnit 5
- **IDE:** IntelliJ IDEA

## 📂 Estrutura do Projeto
O projeto segue a estrutura padrão Maven:
- `src/main/java`: Código-fonte do sistema e modelos de domínio.
- `src/main/resources`: Arquivos de configuração e dados (CSV).
- `src/test/java`: Testes unitários para validação da lógica de negócio.

## 📈 Como Executar
1. Certifique-se de ter o **JDK 21** e o **Maven** instalados.
2. Clone o repositório.
3. Execute o comando `mvn clean install` para compilar e baixar as dependências.
4. Execute a classe `Main` para iniciar o sistema.
